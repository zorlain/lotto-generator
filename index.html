🎱 로또 번호 생성기

동행복권 로또 6/45 전 회차 당첨번호를 분석해서 통계 기반으로 번호를 생성합니다.

🎯 번호 생성

기본 설정으로 5조합을 생성합니다. 조건은 맞춤 번호 생성 탭에서 직접 바꿀 수 있어요.

생성된 번호

⚙️ 맞춤 번호 생성

원하는 조건에 체크하고 값을 정하면, 그 조건에 맞는 번호만 생성됩니다. 체크를 끄면 그 조건은 무시돼요. 아래 "기본 설정"은 누구나 바로 쓸 수 있는 조건이고, "심화 설정"은 조금 더 자세히 파고들고 싶은 분들을 위한 조건입니다.

생성된 번호
🟢 기본 설정 번호 분석이 처음이어도 편하게 쓸 수 있는 조건이에요

직접 고른 번호를 생성되는 조합에 반드시 포함시켜요. 좋아하는 숫자나 기념일처럼 의미 있는 번호가 있다면 아래에서 눌러 선택하세요. 최대 개까지 고를 수 있고, 나머지 자리는 다른 조건에 맞춰 자동으로 채워집니다.

당첨번호 6개 중 홀수(1, 3, 5…)가 몇 개 섞여야 하는지 정해요. 예를 들어 2~4로 두면 홀수 2~4개, 짝수 2~4개로 적당히 섞인 조합만 나옵니다. 실제 당첨번호도 홀수나 짝수 한쪽으로 쏠리기보다 골고루 섞인 경우가 훨씬 많았어요.

뽑힌 번호 6개를 전부 더한 값이 이 범위 안에 들어오게 해요. 이론적으로 합계는 21(1+2+3+4+5+6)부터 255(40~45)까지 가능하지만, 실제 당첨번호는 90~180 사이에 몰리는 경우가 대부분이었습니다. 너무 작거나 너무 큰 합계는 잘 나오지 않아요.

AC값은 번호들 사이 간격이 얼마나 다양한지 보여주는 값이에요(0~10). 예를 들어 2, 4, 6, 8, 10, 12처럼 간격이 일정하면 AC값이 낮고(너무 규칙적인 조합), 간격이 들쭉날쭉할수록 AC값이 높아집니다(더 무작위에 가까운 조합). 실제 당첨번호는 대부분 AC값이 7 이상으로, 규칙성 없이 다양하게 섞인 조합이었어요.

🔬 심화 설정 번호 분석에 익숙한 분들을 위한 고급 필터예요

번호를 1~9, 10~19, 20~29, 30~39, 40~45 다섯 구간으로 나눠서, 각 구간에서 정확히 몇 개씩 나올지 지정해요. 예를 들어 번호가 특정 구간(예: 1~9)에만 몰리지 않게 막고 싶을 때 사용합니다. "자동"으로 두면 그 구간은 몇 개가 나오든 상관하지 않아요.

지금까지 자주 나왔던 번호(HOT)나 반대로 드물게 나왔던 번호(COLD) 중 하나를 골라, 그 번호들이 더 잘 뽑히도록 우대해요. 예를 들어 "HOT 상위 15개"를 선택하면, 자주 나온 번호 15개가 다른 번호보다 이 생성기에서 골라질 확률이 더 높아집니다. 실제 로또 추첨은 완전 무작위라서, 이 설정이 실제 당첨 확률을 높여주지는 않아요.

상위 몇 개 개 번호 우대

마지막으로 당첨된 지 오래됐지만 "슬슬 나올 때가 됐다"고 여겨지는 번호를 우대해요. 예를 들어 15회로 설정하면, 최근 15회차 동안 한 번도 안 나온 번호들이 이 생성기에서 골라질 확률이 더 높아집니다. 다만 로또는 매 회차 완전히 독립적인 추첨이라, 오래 안 나왔다고 실제로 나올 확률이 더 높아지는 건 아니에요.

최근 회 이상 미출현 번호 우대

12, 13처럼 바로 이어지는 숫자("연속번호")를 일부러 넣을지 뺄지 정해요. "반드시 포함"을 고르면 연속번호가 최소 한 쌍은 들어가고, "포함 안 함"을 고르면 연속된 숫자가 전혀 없는 조합만 나옵니다.

적용 방식

번호의 마지막 숫자("끝자리")가 같은 번호들이 너무 많이 겹치지 않게 막아요. 예를 들어 3, 13, 23, 33은 모두 끝자리가 '3'인데, "1개(중복 없음)"를 고르면 6개 번호의 끝자리가 전부 서로 다르게 뽑힙니다.

최대 허용

다른 번호와 같은 회차에 유독 자주 같이 나왔던 "궁합 좋은" 번호들을 우대해요. 예를 들어 7번과 27번이 유난히 자주 함께 나왔다면, 이런 번호들이 이 생성기에서 골라질 확률이 더 높아집니다. 과거에 자주 같이 나왔다고 앞으로도 그럴 확률이 높은 건 아니라는 점은 참고해주세요.

상위 몇 개 개 번호 우대

가장 최근 회차(회)에서 나온 번호와 이번에 생성할 번호가 몇 개까지 겹쳐도 되는지 정해요. 실제로는 바로 전 회차와 아예 안 겹치거나(0개) 1개만 겹치는 경우가 대부분이었습니다.
